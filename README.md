# Healthcare Analytics Dashboard

## Project Overview
This project analyzes healthcare data using Power BI to visualize patient demographics, hospital performance, billing trends, and medical insights.

## SDG Goal
SDG 3: Good Health and Well-Being

## Tools Used
- Power BI
- Excel / CSV Dataset
- Data Visualization Techniques

## Features
- Patient Demographics Analysis
- Medical Condition Insights
- Billing & Insurance Analysis
- Revenue Trends
- Interactive Dashboard Filters

## Dataset Source
https://www.kaggle.com/datasets/prasad22/healthcare-dataset
